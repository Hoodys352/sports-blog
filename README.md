# Sports Blog

## Sports Blog Web App

Sports blog website, includes many different sports !

## How to install

1. Clone the github code
  ```javascript
  git clone https://github.com/Hoodys352/sports-blog.git
  ```
2. Apply migrations if needed ( browser like MS Edge should ask for it )
